Bordio test app

##### Stack:
  - Typescript
  - React
  - React Router
  - i18next
  - Loadable Component (for code splitting)
  - Styled Components

### Installation
```sh
$ git clone https://github.com/epifanovmd/react-template-app.git
$ cd react-template-app
$ yarn
```

### Run
```sh
$ yarn dev
```
